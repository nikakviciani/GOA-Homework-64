programming languages

1. python

python არის მაღალი დონის პროგრამირების ენა, რომელიც შექმნილია 1991 წელს. იგი გამოიგონა გიდო ვან როსუმმა. python არის ინტერპრეტირებული ენა, რომელიც გამოიყენება ვებსაიტების, აპლიკაციების და მონაცემთა ანალიზისთვის. python არის ერთ-ერთი ყველაზე პოპულარული პროგრამირების ენა და აქტუალურია დღემდე.

2. java

java არის მაღალი დონის პროგრამირების ენა, რომელიც შეიქმნა 1995 წელს. იგი გამოიგონა ჯეიმს გოსლინგმა. java არის ობიექტზე ორიენტირებული ენა, რომელიც გამოიყენება ვებსაიტების, აპლიკაციების და თამაშების შესაქმნელად. java არის ერთ-ერთი ყველაზე პოპულარული პროგრამირების ენა და აქტუალურია დღემდე.

3. c++

c++ არის მაღალი დონის პროგრამირების ენა, რომელიც შეიქმნა 1985 წელს. იგი გამოიგონა ბი.ჯი. სტროუფტმა. c++ არის ობიექტზე ორიენტირებული ენა, რომელიც გამოიყენება ვებსაიტების, აპლიკაციების და თამაშების შესაქმნელად. c++ არის ერთ-ერთი ყველაზე პოპულარული პროგრამირების ენა და აქტუალურია დღემდე.
